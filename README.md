## Installation

```bash
$ yarn install
```

## Setup Env

Create a new file called `.env`. Refer `example.env` for the values.

## Running the app

```bash
# development, starts with watch mode.
yarn start:dev

#production
yarn build
yarn start:prod

# Use this in a new terminal to start the microservice.
yarn start:prod:service
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
